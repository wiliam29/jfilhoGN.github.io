Computação Paralela

Nos dias atuais está cada vez mais recorrente o termo computação paralela,que significa uma forma de computação em que vários cálculos podem ser realizados ao mesmo tempo. Este artigo teve como princípio demonstrar de maneira clara e sucinta como são as formas de paralelismo que podem ocorrer, as linguagens de programação que aceitam esse tipo de programação, contextualizar a utilização de GPU e as diferenças entre CPU e GPU e, por fim, um exemplo que demonstra como é o desenvolvimento de aplicações paralelas com foco na programação heterogênea.

    • Segurança da Informação

    A segurança da informação está diretamente relacionada com proteção de um conjunto de informações, no sentido de preservar o valor que possuem para um indivíduo ou uma organização. São características básicas da segurança da informação os atributos de confidencialidade, integridade, disponibilidade e autenticidade, não estando esta segurança restrita somente a sistemas computacionais, informações eletrônicas ou sistemas de armazenamento. O conceito se aplica a todos os aspectos de proteção de informações e dados. O conceito de Segurança Informática ou Segurança de Computadores está intimamente relacionado com o de Segurança da Informação, incluindo não apenas a segurança dos dados/informação, mas também a dos sistemas em si.

    Linguagem de Programação Julia

    Julia é uma linguagem de programação recente, projetada por Jeff Benzanson, Stefan Karpinsnky, Viral Shah e Alan Endelman. Atende requisitos de computação de alto desempenho voltada para área de matemática e engenharia. Esta pesquisa foi feito comparação de desempenho entre a linguagem de programação Julia e Matlab.

    Para realizar esta comparação foi utilizado o problema de equações diferenciais não-lineares ordinárias (EDO).
